# FreeCapT

The free cap table for European founders — multi-jurisdiction, AI-native, free for personal cap tables, $15/mo to share, export, and use AI. Operated by Bifrost Studios (Copenhagen).

## Stack

Next.js 16 (App Router) · TypeScript (strict) · Tailwind CSS v3 · shadcn/ui · Supabase + Drizzle · Vercel.

See `CLAUDE.md` for project context and `docs/` for the full spec.

## Local development

```bash
pnpm install
pnpm dev   # http://localhost:3000
```
